
# 🚀 AtomQuest | Hackathon Submission 1.0

AtomQuest is an AI-driven, high-performance goal management system designed for modern engineering and product teams. It fulfills all mandatory requirements of the BRD and implements several high-value bonus modules.

## 🏗 Architecture & Tech Stack
- **Frontend**: Next.js 15, ShadCN UI, Tailwind CSS.
- **Backend**: Firebase Firestore, Firebase Auth.
- **AI**: Google Genkit + Gemini 2.5 Flash.
- **Analytics**: Recharts for performance trajectory.

## 🛠 Deployment & Setup
1. **GitHub**: Push code to a private or public repo.
2. **Vercel**: Import the project and add environment variables from `.env.example`.
3. **Firebase Console**: 
   - Enable **Email/Password** and **Google** sign-in providers.
   - Add your Vercel URL to **Authorized Domains** in the Auth settings.
4. **Seed**: Click "Initialize Demo Data" on the live dashboard to populate the system.
